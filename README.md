# Linux Administration & Security Projects

This repository contains hands-on Linux projects focused on:
- System administration
- Security hardening
- Log analysis
- User and permission management
- Monitoring and automation

## Projects

### 1. User Management
- Create and manage users using Bash
- Set password policies

### 2. File Permissions
- Secure sensitive files
- Apply least privilege

### 3. Log Analysis
- Analyze /var/log for failed logins
- Detect suspicious activity

### 4. System Monitoring
- CPU, memory, and disk usage alerts

### 5. Network Tools
- Check open ports
- Basic network diagnostics

## Tools Used
- Bash
- Linux CLI
- cron
- grep, awk, sed

## Environment
Tested on Ubuntu 22.04
